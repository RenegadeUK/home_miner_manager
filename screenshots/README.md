## How to Add Screenshots

1. Take screenshots of your running application
2. Save them with the filenames above (PNG format recommended)
3. Place them in this directory
4. Screenshots will automatically display in the main README

## Recommended Size

- Width: 1200-1600px (for good clarity on high-DPI screens)
- Format: PNG with compression
- Consider using tools like TinyPNG to optimize file size
- Dark mode screenshots preferred (matches the modern aesthetic)

## Tips for Great Screenshots

- Use actual data (not lorem ipsum or mock data)
- Show the interface in use (miners running, charts populated)
- Highlight key features (arrows, callouts optional)
- Ensure consistent theme across all screenshots
- Crop to focus on relevant UI elements
- Remove any sensitive information (wallet addresses, IPs)
